Gestor de librerias

Este es un proyecto realizado como parte del cursado de FullStack Developer de EGG Academy.

Fue realizado con Java y Spring para el backend y HTML , CSS y Bootstrap para el frontend. La base de datos utilizada es MySQL

El objetivo de esta web es poder realizar un CRUD que permita crear, listar, editar y eliminar objetos de cualquiera de las entidades disponibles.

En un futuro se agregaran mas funcionalidades
